# Kibela 編集画面用 Chrome拡張

## 概要

- ヘッダー毎に編集用リンクを追加する

[![Image from Gyazo](https://i.gyazo.com/7fb4938b22a2f1c02068cc3f3cf2a016.png)](https://gyazo.com/7fb4938b22a2f1c02068cc3f3cf2a016)

- リンクをクリックすると、編集画面に遷移し、 `だいだい同じ位置` までスクロールします


## インストール
